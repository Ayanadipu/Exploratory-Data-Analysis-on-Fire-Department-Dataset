# 📊 Exploratory Data Analysis on Fire Department Dataset

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a Fire Department dataset to uncover patterns and insights in emergency service calls.

## 🎯 Objectives
- Analyze different types of emergency calls  
- Identify peak hours and trends  
- Explore high-demand locations  
- Visualize data for better understanding  

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## 📂 Dataset
The dataset includes:
- Call type  
- Date and time  
- Location  
- Response details  

## 📊 Analysis Performed
- Data cleaning and preprocessing  
- Handling missing values  
- Time-based analysis (hourly, monthly trends)  
- Location-based analysis  
- Data visualization using graphs  

## 📈 Key Insights
- Identified peak hours of emergency calls  
- Found most frequent call types  
- Highlighted high-demand locations  
- Observed trends over time
